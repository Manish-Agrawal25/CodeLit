# 🚀 CodeLit – AI-Powered Code Editor & Mentor

> **Write. Run. Fix. Learn. All in one place.**

CodeLit is a modern, AI-enhanced coding platform that combines a powerful code editor with intelligent analysis, auto-fixing, and mentorship. Designed for students, developers, and hackathons — it transforms coding into a guided, interactive experience.

**Learn With CodeLit** : [CodeLit](https://codelit-flame.vercel.app/)

---

## ✨ Features

### 🧠 AI-Powered Intelligence

* 🔍 **Explain Code** – Understand what your code does
* ⚡ **Auto Fix** – Detect and fix errors instantly
* 👨‍🏫 **AI Mentor** – Get feedback on strengths, weaknesses & improvements

### 💻 Code Execution

* ▶ **Run Code** – Execute code directly in the browser
* 🌐 Multi-language support

### 🔄 Smart Comparison

* Compare **original vs fixed code**
* Apply fixes with one click ✅

### 🎨 Modern UI/UX

* Monaco Editor (VS Code-like experience)
* Clean dark theme
* Sidebar + toolbar inspired by real IDEs

## ⚙️ Setup Instructions

### 1️⃣ Clone the repo

```bash
git clone https://github.com/your-username/CodeLit.git
cd CodeLit
```

---

### 2️⃣ Run Frontend

```bash
cd frontend
npm install
npm run dev
```

---

### 3️⃣ Run Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
